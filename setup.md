# DocMind Setup (Mac)

This repo contains:

- `frontend/` (Next.js + TypeScript)
- `backend/` (.NET minimal API)

Goal: get local dev running with:

- Frontend: http://localhost:3000
- Backend: http://localhost:5237 (or whatever port dotnet prints)

---

## 0) Prereqs

- macOS (Intel or Apple Silicon)
- Command Line Tools (if Homebrew install asks):
  ```bash
  xcode-select --install
  ```
